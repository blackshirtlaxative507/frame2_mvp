## 📊 About frame²

frame² is a desktop app that helps you understand sports games. It focuses on baseball, especially MLB and Red Sox matches. It shows why a game’s result changed. The app breaks results into two parts: process and variance. This means it separates what happened because of skill or plan from random chance.

You don’t need to know anything about coding to use this app. It runs on Windows and uses data from real games. The app uses simple visual displays to explain complex stats called sabermetrics. It brings useful insights right to your screen.
